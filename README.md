# Beauty Studio Hub

Маркетплейс бьюти-услуг для клиентов и владельцев салонов.

## Технологии

### Backend
- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- JWT Auth

### Frontend
- React.js (Vite)
- Tailwind CSS
- Lucide-react

---

## 1. Клонирование / создание проекта

Создайте две папки:

- backend
- frontend

---

## 2. Запуск PostgreSQL через Docker

В корне проекта выполнить:

```bash
docker compose up -d
